## Description

This repository contains C programs implementing various sorting algorithms, such as Bubble Sort, Insertion Sort, Selection Sort, and Quick Sort. The code includes functions for printing arrays and doubly linked lists, as well as the implementation details of each sorting algorithm.

## Files

- `sort.h`: Header file containing structure definitions and function prototypes.
- `print_array.c`: Function to print an array of integers.
- `print_list.c`: Function to print a doubly linked list of integers.
- `0-bubble_sort.c`: Bubble Sort implementation.
- `1-insertion_sort_list.c`: Insertion Sort implementation for doubly linked lists.
- `2-selection_sort.c`: Selection Sort implementation.
- `3-quick_sort.c`: Quick sort implementation.
- `0-O`, `1-O`, `2-O`, `3-O`: Files that contain the of their corresponding Soting algorithims.

## Functions

- `print_array`: Prints an array of integers.
- `print_list`: Prints a doubly linked list of integers.

## Sorting Algorithms

- `bubble_sort`: Bubble Sort algorithm for arrays.
- `insertion_sort_list`: Insertion Sort algorithm for doubly linked lists.
- `selection_sort`: Selection Sort algorithm for arrays.
- `quick_sort`: Quick Sort algorithm for arrays.

## Usage

Include the `sort.h` header file in your C program and use the provided functions for printing arrays and doubly linked lists. Choose the appropriate sorting algorithm for your needs.

## Big O Notation

- **Bubble Sort**: O(n^2)
- **Insertion Sort**: O(n^2)
- **Selection Sort**: O(n^2)
- **Quick Sort**: O(nlog(n))
